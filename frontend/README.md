# Визуализатор Сетевой Топологии (Network Topology Viewer)

Интерактивная панель мониторинга и визуализации сетевой топологии в реальном времени для систем управления сетью (NOC). Проект отображает узлы сети, линии связи, передает метрики трафика и выводит лог сетевых инцидентов.

## 🛠 Технологический стек

### Frontend (Клиентская часть)
*   **Ядро:** React 19, TypeScript, Vite
*   **Стилизация:** SCSS Modules (`*.module.scss`) и CSS-переменные для глобальной темы
*   **Библиотека компонентов:** Ant Design (antd v6)
*   **Интерактивная карта:** React Flow (`@xyflow/react` v12) с оптимизацией рендеринга при обновлении данных телеметрии (исключены лишние циклы обновления во время перемещения элементов/зума)
*   **Виртуализация списков:** React Virtuoso (для лога инцидентов/логов в `AlertsPanel`)
*   **Управление состоянием:** Redux Toolkit & RTK Query (для работы с WebSocket-подключениями к бэкенду)

### Backend (Серверная часть)
*   **Ядро:** Node.js, Express, TypeScript
*   **Симуляция реального времени:** WebSockets (`ws`) — рассылка событий топологии, загрузки интерфейсов устройств и инцидентов безопасности

## 🚀 Установка и запуск

### Пререквизиты
Убедитесь, что у вас установлена Node.js (рекомендуется версия 18 или новее) и менеджер пакетов `npm`.

### 1. Установка всех зависимостей
Для установки всех пакетов (в корне, во frontend и в backend) выполните команду в корневом каталоге проекта:
```bash
npm run install:all
```

### 2. Запуск в режиме разработки (Dev)
Запуск frontend и backend в параллельном режиме (с автоперезагрузкой при изменении файлов):
```bash
npm run dev
```
*   **Интерфейс (Frontend):** будет доступен по адресу `http://localhost:3000/` (с поддержкой Vite Dev Server).
*   **Сервер симуляции (Backend):** запустится на порту `3001` (WebSocket на `ws://localhost:3001`).

### 3. Сборка для Production
Для компиляции и сборки оптимизированного клиентского бандла:
```bash
npm run build --prefix frontend
```
Собранные файлы будут помещены в директорию `frontend/dist/`.

## 📁 Структура проекта

*   `backend/` — исходный код WebSocket-сервера симуляции сетевой активности.
*   `frontend/` — исходный код клиентского React-приложения.
    *   `public/` — статические файлы (включая входную точку `index.html` и векторные иконки).
    *   `src/` — исходный код компонентов, страниц и стора.
        *   `entities/` — сущности (устройства, соединения, алерты).
        *   `widgets/` — крупные блоки интерфейса (карты топологии, панели мониторинга).
        *   `pages/` — страницы приложения.
        *   `app/` — глобальные настройки, стили и Redux Store.
